Anonymous chatBox - Night

Anonymous ChatBox est un plugin JQuery qui permet d'intégrer une chatBox dans une page html, il fonctionne en javascript et il est asychrome, cela veux dire qu'il peut fonctionner sans le rafraichissement de la page et ne bloque pas la page lors de son fonctionnement.

Pour les dévellopeurs :

Le code html(seulement html !) du chat se trouve dans "anonymousChatBox.html"
La page php qui traite les données envoyées par le chat est "anonymousChatBox.php" Le javascript du plugin se trouve dans "anonymousChatBox.js"
Les différents styles css se trouvent dans le fichier "css".
Le refresh a lieu toutes les 4 secondes, pour modifier cette durée contactez moi ou changer la dans le fichier javascript

Comment l'intégrer?

avec la méthode .anonymousChatBox(); /!/ Attention cela demande d'intégrer jquery /!/.
Pour cela il suffit d'intégrer le code suivant dans votre page html avant anonymousChatBox: <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.3.1/jquery.min.js"></script>
Secondement il suffit d'intégrer le chat: C'est simple, il suffit de mettre le script juste après le jquery:<script type="text/javascript" src="anonymousChatBox.js"></script> Vous devriez éxécuter votre code comme ceci:


$(function()
{
$('sel').anonymousChatBox('style', 'urlChat', 'anonymousChatBoxFiles', 'langue')
}


sel:selecteur css (tuto : 'https://developer.mozilla.org/fr/docs/Web/CSS/S%C3%A9lecteurs_CSS')
.anonymousChatBox: methode pour faire afficher le chat
style: vous avez plusieurs choix du style:

  • -night, par défaut un mode nuit
    • Essayer
  • -light, mode jour
    • Essayer
  • -dark, mode jour avec du noir
    • Essayer
  • -bigchef, bleu/blanc, aux couleurs du créateur
    • Essayer
  • -orange, orange/blanc
    • Essayer
  • -secret, a vous de le trouver !(celui qui le trouve en premier à le droit à l'implantation gratuite sur son site.)

    • A vous de trouver !


urlChat:url du chat, par défaut: 'derniersMessages.htm'
!/! Attention, si vous mettez pas un type de requête (http par exemple), le chat s'actualiseras pas en direct.(il est pas mit par défaut)
anonymousChatBoxFiles:, l'url du fichier "anonymousChatBox", si vous indiquez pas alors que votre fichier est en dehors de se dossier, sa ne fonctionne pas.
